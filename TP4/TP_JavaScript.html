TP2 : LES VARIABLES ET OPÉRATIONS DE LECTURE

Objectifs

  • Savoir déclarer une variable.
  • Utiliser le bon type de variable (int, float, etc.).
  • Effectuer des saisies utilisateurs.
  • Comprendre le mécanisme du cast.

Exercice 1 : Déclaration de variables

Indiquer si ces variables sont valides ou pas en Java (type int) :

  • age
  • !Ex
  • R79JH3
  • Revenu98
  • invalide
  • élu
  • long
  • 3v

Exercice 3 : La Classe Scanner

Cette classe permet de lire les entrées clavier (saisie utilisateur).

Importation : import java.util.Scanner;

Déclaration de l'objet : Scanner sc = new Scanner(System.in);

Tests de saisie

Ordre de saisie à tester : byte, short, int, long, float, double, boolean.

Exercice 4 : Le mécanisme du CAST

Gestion de 5 notes (n1 à n5) de type byte.

  1. Saisir et afficher les 5 notes.
  2. Calculer la somme dans une variable de type byte (nécessite un cast)
