1. Structure de base d'un programme

[cite_start]

Lorsque le code se trouve dans un fichier séparé (.js), la structure est la suivante[cite: 1]:

/* zone d'en-tête */

/* déclaration des variables globales */
[cite_start]let variable1; [cite: 2]

/* définition des fonctions */
[cite_start]function maFonction() { ... } [cite: 3]

/* CORPS DU PROGRAMME */
[cite_start]instruction1; [cite: 3]
[cite_start]

Il est conseillé de déclarer les variables en début de script pour une meilleure lisibilité[cite: 4].

Intégration dans une page Web

[cite_start]

Dans un document HTML, on utilise la balise <script>[cite: 9]:

  • Dans le <head> [cite_start]: Pour les variables globales et les fonctions[cite: 9].
  • Dans le <body> [cite_start]: Pour les instructions directes[cite: 9].

3. Déclaration de Variables et Constantes

[cite_start]

JavaScript est un langage faiblement typé[cite: 20]. [cite_start]Contrairement au langage C, on ne précise pas le type à la déclaration ; il est défini lors de l'utilisation[cite: 24].

Mots-clefs de déclaration

  • let : Déclare une variable locale au bloc d'instruction { }. [cite_start]C'est le choix recommandé[cite: 46, 49].
    • [cite_start]
  • var : Déclare une variable globale (sauf dans une fonction)[cite: 45].
    • [cite_start]
  • const : Déclare une constante qui doit être initialisée dès sa création et dont la valeur ne changera pas[cite: 51, 52].

Règles de nommage

[cite_start]

Les noms peuvent contenir a-z, A-Z, 0-9 (pas au début) et _[cite: 36]. [cite_start]Attention : le langage est sensible à la casse (Poste_1 est différent de poste_1)[cite: 40, 41].

5. Opérateurs

Arithmétiques

[cite_start]

+, -, *, /, % (modulo), ** (puissance)[cite: 81].

Attention : JavaScript tente toujours de retourner un résultat. [cite_start]Si l'opération est impossible (ex: 5 / "test"), il retourne NaN (Not A Number)[cite: 84, 86].

Logiques et Comparaison

    [cite_start]
  • == (égal), != (différent), <, >, <=, >=[cite: 100].
    • [cite_start]
  • && (ET), || (OU), ! (NON)[cite: 99, 100].

Incrémentation et Affectation

    [cite_start]
  • i++ : Post-incrémentation (i = i + 1)[cite: 103].
    • [cite_start]
  • i += 5 : Ajout combiné (i = i + 5)[cite: 102].
  • Affectation conditionnelle : k = (i > j) ? [cite_start]10 : -10; (Si vrai alors 10, sinon -10)[cite: 107, 109].

[cite_start]

Pour dialoguer avec l'utilisateur dans un navigateur[cite: 109, 110]:

  • window.alert("Message"); : Affiche une boîte d'alerte[cite: 110].
    • [cite_start]
