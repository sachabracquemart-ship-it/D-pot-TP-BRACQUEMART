EXERCICE 2 : Le langage SQL

1. Le langage SQL

1.1. Définition

Le langage SQL (Structured Query Language ou Langage de requêtes structuré) est un langage comprenant à lui seul l'ensemble des ordres nécessaires à la création et à la gestion d'une base de données relationnelle. Il a été créé en 1986.

1.2. Les différentes normes utilisées

Il existe aujourd'hui plusieurs normes relatives à l'utilisation du langage SQL :

  • La norme SQL 89 qui est encore très utilisée actuellement.
  • La norme SQL 92.
  • Aujourd'hui SQL 2011.

Bien qu'il existe des normes il peut y avoir des différences entre SGBDR (PostgreSQL, etc.) :

  • "Le développement de PostgreSQL respecte le standard en vigueur..."
  • "SQL-92 définit trois niveaux de conformité..."
